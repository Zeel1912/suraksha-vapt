SURAKSHA Vulnerable Test Environment
===================================

This is a deliberately vulnerable web application created for educational purposes.

WARNING: This application contains intentional security vulnerabilities and should NEVER be exposed to the internet or used in production environments.

Vulnerabilities Included:
- SQL Injection
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Directory Traversal
- Broken Authentication
- Insecure Cookies
- Security Misconfiguration

Use this environment only with the SURAKSHA VAPT platform for learning cybersecurity concepts.

For more information, visit: https://github.com/Zeel1912/suraksha-vapt
